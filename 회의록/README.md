# 🏴‍☠️ 해적 수배서 팀 소개 랜딩페이지

> **과제 목표:** 도발적이고 개성 있는 방식으로 팀원을 소개하는 랜딩페이지 제작

---

## 📋 프로젝트 개요

본 프로젝트는 단순한 자기소개를 넘어, **보는 사람이 임팩트를 받을 수 있는** 도발적인 팀 소개 랜딩페이지를 목표로 제작되었습니다.

- **팀원:** 이건후, 이원영, 한영관
- **지도교수:** 김남주 교수님

---

## 💡 아이디어 회의

랜딩페이지의 컨셉을 정하기 위해 다양한 아이디어가 논의되었습니다.

| 아이디어 | 내용 |
|---|---|
| 📖 나무위키 스타일 | 팀원 3명 각각의 나무위키식 설명 문서 제작 |
| 🧠 교수님 척수 흡수 성장기 | 교수님들의 지식을 흡수하며 성장하는 모습을 템플릿화 |
| 📈 우량주 주식 차트 | "우리는 우량주다!" 컨셉의 주식 차트 형식 소개 |
| **🏴‍☠️ 원피스 해적 수배서** ✅ | **원피스의 해적 수배서 형식으로 팀원 소개** ← **최종 채택** |

---

## 🎨 제작 방식

### 1단계 – 캐릭터 선정 및 사진 합성

팀원 각자가 원하는 **원피스 캐릭터**를 선택하고, 본인의 증명사진과 합성하여 수배서 앞면 이미지를 제작하였습니다.

| 팀원 | 원피스 캐릭터 |
|---|---|
| 이원영 | 쵸파 (Tony Tony Chopper) |
| 이건후 | 죠로 (Roronoa Zoro) |
| 한영관 | 상디 (Sanji) |

### 2단계 – 수배서 카드 구성

각 수배서는 **앞면(합성 사진)** 과 **뒷면(개인 정보)** 으로 구성되며, 뒷면의 정보는 코믹하게 작성되었습니다.
